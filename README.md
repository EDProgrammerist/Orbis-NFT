# Orbis NFT — Next.js 16 

## Tech Stack

- **Next.js 16.2** (App Router, Turbopack)
- **React 19**
- **Tailwind CSS v4** (CSS-first config via `@theme`)
- **TypeScript**
- **lucide-react** for icons


## Getting Started

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # Production build
```
